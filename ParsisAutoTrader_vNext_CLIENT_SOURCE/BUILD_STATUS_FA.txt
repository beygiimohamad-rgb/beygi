این بسته سورس واقعی نسخه جدید و خط ساخت Windows را دارد، اما در محیط فعلی Linux فایل WinUI 3 و Inno Setup قابل کامپایل و اجرای نهایی نیست.
برای تولید Setup.exe واقعی، Workflow موجود باید روی GitHub Actions windows-latest یا یک Windows x64 اجرا شود.
هیچ Setup جعلی که هنگام نصب پروژه را Build کند در این بسته وجود ندارد. Setup نهایی فقط فایل‌های از قبل کامپایل‌شده را نصب می‌کند.
